Simple A + B Problem

Description

请计算两个整数的和并输出结果。

注意不要有不必要的输出，比如"请输入 a 和 b 的值: "，示例代码见隐藏部分。

Input

两个用空格分开的整数.

Output

两数之和

Sample Input 1 

1 1

Sample Output 1

2

Hint

C 语言实现:

#include <stdio.h>    
int main(){
    int a, b;
    scanf("%d%d", &a, &b);
    printf("%d\n", a+b);
    return 0;
}

Java 实现:

import java.util.Scanner;
public class Main{
    public static void main(String[] args){
        Scanner in=new Scanner(System.in);
        int a=in.nextInt();
        int b=in.nextInt();
        System.out.println((a+b));  
    }
}

JS 实现:

const readline = require('readline');
const rl = readline.createInterface({ input: process.stdin });
rl.on('line', (input) => {
  if (input === '') {
    return rl.close();
  }
  const [a, b] = input.split(' ').map(Number)
  console.log(a + b);
});
